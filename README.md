# Проект 5: Базовый Javascript "Mesto"

Функционал: 
* добавление и удаление карточек (мест) с подписью и картинкой; 
* редактирование профиля;
* Отметка лайком любимых мест
* Увеличение фото.

Используемые технологии:
* Javascript, HTML, CSS
